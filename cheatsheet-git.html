Git and GitHub Cheat Sheet

Ouctober 31, 2021 by Conrado Almeida

Create a Repository
$ git ini [project name] From scratch --Create a new local repository
$ git clone my_url Download from an existing repository
Observe your Repository
$ git status List new or modified files not yes commited
$ git diff Show the changes to files not yes staged
$ git diff --cached Show changes to staged files
$ git diff HEAD Show all staged and unstaged files changes
$ git diff commit1 commit2 Show the changes between two commits ids
$ git blame [file] List the change dates and authors for a file
$ git show [commit]:[file] Show the file changes for a commit id and/or file
$ git log Show full change history
$ git log -p [file/directory] Show change history for file/directory
Working with Branches
$ git branch List all local branches
$ git branches -av List all branches, local and remote
$ git checkout my_branch Switch to a branch and update
$ git branch new_branch Create a new branch called new_branch
$ git branch -d new_branch Delete the branch called my_branch
$ git checkout branch_b Merge branch_a into branch_b
$ git checkout branch_b
$ git tag my_tag Tag the current commit
Make a Change
$ git add [file] Stages the file, ready for commit
$ git add . Stafe all changed files, ready for commit
$ git commit -am "message" Commit all your tracked files to versioned history
$ git reset [file] Unstages files, keeping the file changes
$ git reset --hard Revert everything the last commit
Synchronize
$ git fetch Get the latest changes from origin (no merge)
$ git pull Fetch the latest changes from origin and merge
$ git pyll --rebase Fetch the latest changes from origin and rebase
$ git push Push local changes to the origin
